# Оптимизации pipeline_v55_final.py (01.07.2026)

## Дедупликация
- **Проблема**: Ложные срабатывания `fuzzy_dedup` (например, `BRAND BUILDING GROUP` и `BUILDING BRAND`).
- **Решение**: Сортировка токенов в `normalize()` и проверка длины названий (отклонять только если длина отличается ≤ 20%).

## Скоринг
- **Проблема**: Высокий скор отклоняет реальные компании (например, `BARRY CALLEBAUT`).
- **Решение**: Белый список известных компаний и снижение порога высокого скора с 40 до 50.

## Фильтрация телефонов
- **Проблема**: Маскированные телефоны (`1111111111`) проходят валидацию.
- **Решение**: Проверка на `1111111111`, `0000000000`, `1234567890`.

## Обогащение контактов
- **Проблема**: Серые зоны (30-50 скор) требуют ручной проверки.
- **Решение**: Добавить модуль `web_search` для обогащения email/phone/website.

- **Тестовые данные**: `input_test.csv` (18 записей).
- **Результаты**:
