## Ключевые выводы

### Лучший источник телефонов
**zachestnyibiznes.ru + sbis.ru** — hit rate **96%** для записей с ИНН (317 из 328).
- zachestnyibiznes.ru: парсить `tel:` ссылки на карточке компании
- sbis.ru: парсить regex на странице контрагента
- Паузы 2-4 сек, ротация User-Agent

### НЕ использовать
- **list-org.com** — блокирует IP после ~40 запросов (timeout на уровне соединения)
- **Serper для поиска сайтов** — много мусора (каталоги выставок, агрегаторы, PDF)
- **Парсинг сайтов** — малоэффективен (только ~11 реальных сайтов из 1714)

### Артефакты sbis.ru
- `+72202269599` (38 записей), `+71984401665` (21 запись) — номера хостинга
- После обогащения через sbis.ru **ВСЕГДА** очищать артефакты
- Очистка ТОЛЬКО по `moysklad_id`, НЕ по `phone=eq.`

### Чёрный список доменов
Обязателен для фильтрации. Без него 70%+ результатов — мусор.
Топ мусорных доменов: torgbox.ru, prod-expo.ru, icatalog.expocentr.ru, vk.com

### Supabase SDK
**Зависает на длительных операциях** — всегда использовать REST API через `requests`.

### Supabase DELETE/PATCH
- HTTP 204 = успех (No Content, тело пустое для `Prefer: return=minimal`)
- HTTP 400 для DELETE без фильтра — нужно условие `?field=value`
- `in.()` в DELETE работает для батч-удаления
