# Маппинг рубрик → категории

Источник: 343 уникальных рубрики из поля `notes` в temp_clients.

## Целевые

### Кондитерские изделия
Оборудование для кондитерской промышленности, производства шоколада, конфет, пастилы, мармелада, карамели, вафель, печенья, тортов, пирожных. Глазировочное оборудование.

### Глазури
Глазировочное оборудование, кондитерские полуфабрикаты, coating.

### Мороженое
Оборудование для производства мороженого, глубокой заморозки, криогенная/шоковая заморозка, морозильные аппараты.

### Хлеб/Пекарни
Хлебопекарное оборудование, мини-пекарни, изготовление теста, автоматизированные линии хлебобулочных.

### Молочные десерты
Переработка молока, производство сыров, кисломолочных культур, йогуртов, творожных изделий, детского питания. Ингредиенты для молочной промышленности.

### Дистрибьюторы сырья
Ингредиенты для хлебопекарной/кондитерской/молочной промышленности, комплексные смеси, моноингредиенты, пищевые добавки.

### Ингредиенты
Специи, пряности, ароматизаторы, красители, консерванты, какао-продукты, лецитин.

## Не целевые

### Оборудование (не производство)
Мясо/птица/рыба, убой скота, промышленные роботы, складские холодильники, климат/вентиляция, теплообменники, насосы, транспортное оборудование.

### Упаковка
Упаковочное оборудование, материалы, этикетирование, гофрокартон.

### Инфраструктура
Проектирование, строительство, автоматизация, санитария, моющие средства, наливные полы, трапы, дезинфекция.

### Прочее
СМИ, порталы, оболочки, спецодежда, инвентарь, запчасти, «Российский экспортер» (метка).
