# Анализ какао-порошка (14.06.2026)

## Ошибки моделей

### Примеры невалидного JSON
1. **Markdown-обёртки**:
```json
```json
{
  "группы": ["Кондитерские фабрики"]
}
```
```

2. **Экранированные переносы**:
```json
{
  "группы": ["Кондитерские\nфабрики"]
}
```

3. **Синтаксические ошибки**:
```json
{
  "группы": ["Кондитерские фабрики",]
}
```

## Ключевые слова для поиска производителей
- **Кондитерские фабрики**: "какао-порошок", "шоколадная глазурь", "кондитерские изделия", "глазурь для выпечки".
- **Производители ЗОЖ**: "здоровое питание", "веганские десерты", "протеиновые батончики", "органические продукты".

## Паттерны для группировки сегментов
- **Кондитерские фабрики**:
  - Крупные производители (объём > 1000 тонн/год).
  - Малые производители (объём < 1000 тонн/год).
- **Производители ЗОЖ**:
  - Протеиновые батончики.
  - Веганские десерты.
  - Органические продукты.

## Пример объединения данных
```python
from collections import defaultdict

# Данные из основного анализа
main_data = {
  "Какао-порошок": {
    "Кондитерские фабрики": {
      "Крупные производители": ["фабрика1", "фабрика2"],
      "Малые производители": ["фабрика3"]
    }
  }
}

# Данные из сегмента ЗОЖ
health_data = {
  "Какао-порошок": {
    "Производители здорового питания": {
      "Протеиновые батончики": ["бренд1", "бренд2"],
      "Веганские десерты": ["бренд3"]
    }
  }
}

# Объединение
merged = defaultdict(dict)
for data in [main_data, health_data]:
    for product, groups in data.items():
        for group, subgroups in groups.items():
            merged[product][group] = subgroups
```
