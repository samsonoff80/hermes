# Consilium HTML Catalog Analysis (28.06.2026)

## Use Case: Analyze HTML to Extract Parsing Config

Beyond standard audit/analysis tasks, Consilium can analyze HTML pages and return structured configs for parsing.

### Prompt Template
```
Analyze this HTML catalog page and determine how to parse it for company data.

URL: {url}

HTML (first 30KB):
```html
{html[:25000]}
```

Return ONLY valid JSON:
{{
  "data_structure": "table" | "card" | "spa",
  "parsing_method": "requests" | "playwright",
  "selectors": {{
    "name": "CSS selector for company name",
    "country": "CSS selector for country (or empty string)",
    "card_url": "CSS selector for company page link (or empty string)",
    "card": "CSS selector for company card/item (or empty string)"
  }},
  "has_pagination": true | false,
  "pagination_type": "offset" | "page" | "url_pattern" | "",
  "max_pages": number,
  "phone_on_list": true | false,
  "email_on_list": true | false,
  "website_on_list": true | false,
  "needs_detail_page": true | false,
  "detail_selectors": {{
    "phone": "CSS selector for phone on detail page",
    "email": "CSS selector for email on detail page",
    "website": "CSS selector for website on detail page",
    "city": "CSS selector for city on detail page"
  }},
  "company_count_estimate": number,
  "confidence": 0.0-1.0,
  "notes": "brief notes about the site structure"
}}
```

### Key Rules for Prompt
- "table" if companies in HTML tables (`<tr><td>`)
- "card" if companies in div cards or list items
- "spa" if content loads via JavaScript (no static HTML)
- "requests" if HTML contains all data
- "playwright" if SPA or requires JS rendering
- Selectors must work with BeautifulSoup (Python library)

### Integration Pattern
```python
# consilium_brain.py
async with aiohttp.ClientSession() as session:
    result = await consilium_ask(session, prompt, models=models)

# Parse JSON from response
best = result.get("best", "")
config = extract_config_from_response(best)  # regex + json.loads fallback
```

### Real Example (28.06.2026)
**Input**: `https://foodsuppliers.ru/sitemap/list` (30KB HTML)
**Consilium output**:
```json
{
  "data_structure": "card",
  "parsing_method": "requests",
  "selectors": {
    "name": ".abc-box__list li a",
    "card_url": ".abc-box__list li a",
    "card": ".abc-box__list li"
  },
  "has_pagination": true,
  "pagination_type": "url_pattern",
  "confidence": 0.80,
  "company_count_estimate": 192
}
```
**Result**: 192 companies parsed successfully with config generated by 2 models.
