# Шаблон анализа продукта (Product Analysis)

## Структура JSON
```json
{
  "product": "Название продукта",
  "subgroups": [
    "производство [подгруппа 1]",
    "производство [подгруппа 2]",
    "производство [подгруппа 3]"
  ],
  "groups": [
    "Кондитерские изделия",
    "Молочные продукты",
    "Хлеб/Пекарни",
    "Здоровое питание",
    "Снеки"
  ],
  "keywords": [
    "фабрика [тип 1]",
    "производитель [тип 2]",
    "завод [тип 3]"
  ],
  "exclude_keywords": [
    "конкуренты по [продукту]",
    "импортеры [продукта]"
  ],
  "models_ok": "3/3",
  "needs_review": false
}
```

## Рекомендации по использованию
1. **Запуск**: Использовать готовый скрипт `analyze_product.py` (не писать код).
2. **Параллелизм**: Запускать по 3 продукта одновременно для ускорения.
3. **Обработка лимитов**: Если OpenRouter возвращает 429, переключаться на 3 модели (Mistral, Groq, Cloudflare).
4. **Сохранение**: Результаты автоматически сохраняются в `data/last_analysis.json`.
